  • 선수ID
  • 이름
  • 등번호
경기 데이터 (Match Data)
  • match_id: 경기 고유 ID
  • date: 경기 일시
  • venue: 경기장 정보 (유형
  • home_team / away_team: 홈/원정팀
  • final_score: 최종 스코어
이벤트 데이터 (Event Data)

이벤트 공통 속성

event_id: 이벤트 고유 ID

timestamp: 발생 시간 (millisecond)

set_number: 세트 번호

rally_number: 랠리 번호

score_state: 점수 상태 (home

position_x, position_y: 위치 좌표

이벤트 유형별 상세

서브 (Serve)

  • type: floater/jump/jump_float
  • result: ace/fault/in_play
  • zone: 1-6, speed, trajectory

공격 (Attack)

  • type: spike/tip/roll_shot
  • result: kill/blocked/error/in_play
  • position, tempo, direction

블로킹 (Block)

  • type: solo/assist/touch
  • result: stuff/touch/miss
  • players_involved, reach_height

디그 (Dig)

  • type: dig/emergency/pancake
  • result: perfect/good/overpass/error
  • attack_type_received, body_part

리시브 (Receive)

  • type: forearm/overhand
  • result: perfect/good/poor/ace
  • serve_type_received, target_accuracy

토스 (Set)

  • type: regular/back/jump
  • tempo, height, target_position
  • result: hittable/poor/error

4. 상세 구현 기능

4.1 환경 적응형 AI 모델 구현

구현 항목 상세 내용 기술 스택
코트 라인 설정 UI 4개 코너 포인트 기반 인터랙티브 앵커링 및 호모그래피 자동 계산 React, Canvas API
프리셋 관리 시스템 구장별 카메라 뷰포트 저장 및 불러오기 기능 IndexedDB, REST API
온라인 캘리브레이션 KLT Optical Flow 기반 특징점 추적 및 RANSAC 아웃라이어 제거 OpenCV, NumPy
Auto-Correction 칼만 필터를 통한 카메라 위치/회전 실시간 추정 및 노이즈 저감 Kalman Filter
HSV 색공간 변환 조명 의존적 RGB 대신 HSV 색공간 채택으로 조명 변화 대응 OpenCV, PIL
Dynamic Thresholding 구장 바닥면과 라인 색상 대비 극대화를 위한 적응형 임계값 Histogram Analysis
노이즈 필터 라인 규격 외 파편화된 객체 제거 및 타 종목 라인 필터링 Morphological Ops
앙상블 추론 복수 모델 결합을 통한 안정적 예측 Weighted Voting

  • 시스템 간 인터페이스 검증

  • End-to-End 시나리오 검증
  • 성능 부하 테스트
  • 보안 취약점 점검
