알고리즘 1: 호모그래피(Homography) 자동 계산

4개 코너 포인트 기반 좌표 변환 행렬 계산

수학적 정의

H = 3x3 변환 행렬

[x'] [h11 h12 h13] [x]

[y'] = [h21 h22 h23] [y]

[1 ] [h31 h32 h33] [1]

4개의 대응점으로부터 8개의 방정식을 구성하여 H 행렬의 8개 자유도를 결정

입력 데이터

  • - 영상 좌표계: 4개 코너 포인트 (픽셀 단위)
  • - 월드 좌표계: 배구 코트 실제 치수 (9m x 18m)
  • - 해상도: 1920x1080 또는 3840x2160

구현 단계

1

포인트 수집

사용자가 UI에서 4개 코너 클릭 시 좌표 저장

2

DLT 알고리즘 적용

Direct Linear Transform으로 H 행렬 초기값 계산

3

비선형 최적화

Levenberg-Marquardt로 재투영 오차 최소화

4

검증 및 적용

재투영 오차 2px 이내 확인 후 적용

  • 중앙선-사이드라인 교차점 (2개)

알고리즘 4: 칼만 필터 기반 노이즈 저감

카메라 위치/회전의 시계열 추정 및 노이즈 필터링

상태 벡터 정의

State Vector (12D):

x = [tx, ty, tz, // 위치 (3D)

rx, ry, rz, // 회전 (3D)

vtx, vty, vtz, // 속도 (3D)

vrx, vry, vrz] // 각속도 (3D)

카메라의 6DOF 자세와 그 변화율을 동시에 추정

칼만 필터 루프

Predict: 이전 상태로부터 현재 상태 예측
Measure: 특징점 추적 결과로부터 관측값 획득
Update: 예측값과 관측값을 융합하여 상태 보정
Output: 노이즈가 제거된 안정적인 카메라 자세

칼만 필터 적용 전/후 성능 비교

적용 전 (Raw Tracking)

  • - 위치 노이즈: ±5px
  • - 프레임 간 지터링 발생
  • - 급격한 변화 시 불안정

적용 후 (Kalman Filtered)

  • - 위치 노이즈: ±0.5px
  • - 부드러운 연속 추적
  • - 급격한 변화 대응 가능
